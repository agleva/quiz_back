# Readme
# Versão do Node
18.16.0
# Instalação das dependências
Faça npm install
# Preparando o ambiente
### DB 
execute no terminal o comando docker-compose up mysql (Em breve utilizaremos o docker-compose up para criar a aplicação completa no docker)
#### Obs: necessário ter docker instalado e aberto. 
# Iniciando o ambiente
npm run start:dev
# Versionamento
Para versionar crie uma branch para cada issue/task
### Crie a branch sempre a partir da develop atualizada
git branch nome-da-branch
### Mude de branch
git checkout nome-da-branch
#### Após finalização da issue/task enviar como mensagem do commit
git commit -m "mensagem do commit. Fixes #numero da issue"
